Classes vs IDs

The best practices associated with using Classes vs. IDs


Classes are good for using when:

  • You want to use the same class on different elements
  • You want to style elements with multiple different classes
  • You want to have a little bit more flexibility in terms of overriding later

IDs are good for using when:

  • Wanting distinct element identification
  • Wanting to use those particular elements for jump links

Deciding on whether to use a class or ID for your elements really comes down to what you want to get out of it.
As mentioned in the points above there are reasons for using either or.
